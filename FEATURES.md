# Features

## MVP Features
- Character part selection (body, face, eyes, hair, etc.)
- Live template preview
- User text prompt
- AI image generation
- Template lock strength (hard / soft)
- Anime-style character builder
- Modular anime-specific assets
- Template locking for anime anatomy
- AI rendering optimized for anime prompts

## Phase 2 Features
- Preset characters
- Multiple art styles
- Save & load characters
- Export images

## Experimental
- Batch generation
- Variant generation
